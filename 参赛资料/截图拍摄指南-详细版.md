# HarmonyCraft 截图拍摄详细指南

> 本指南提供两种截图方案：**方案A-模拟器截图**（推荐，画质稳定）和 **方案B-真机截图**（更真实）

---

## 方案A：DevEco Studio模拟器截图（推荐）

### 第一步：启动模拟器

1. 打开 DevEco Studio
2. 菜单栏 → **Tools → Device Manager**
3. 选择 **Local Emulator** → 点击 **New Emulator**
4. 选择设备：Phone（建议选 **Huawei Phone** 或默认手机模板）
5. 系统镜像：选择 **API 12 (6.0.2) 或 API 22**
6. 点击 **Next → Finish**，等待模拟器启动（首次约2-3分钟）

### 第二步：安装应用并准备数据

```bash
# 1. 先构建hap
cd ~/DevEcoStudioProjects/HarmonyCraft
DEVECO_SDK_HOME=/Applications/DevEco-Studio.app/Contents/sdk \
/Applications/DevEco-Studio.app/Contents/tools/hvigor/bin/hvigorw \
  --mode module -p product=default -p module=entry@default \
  assembleHap --no-daemon

# 2. 安装到模拟器
hdc install entry/build/default/outputs/default/entry-default-unsigned.hap

# 3. 启动应用
hdc shell aa start -a EntryAbility -b com.example.harmonycraft
```

## 批量截图自动化脚本

如果你已经准备好了所有数据，可以把这个脚本保存为 `screenshot.sh`，按顺序执行：

```bash
#!/bin/bash
# screenshot.sh - HarmonyCraft 批量截图脚本

SAVE_DIR="$HOME/Desktop/HarmonyCraft截图"
mkdir -p "$SAVE_DIR"

echo "=== HarmonyCraft 截图工具 ==="
echo "请按顺序切换到以下页面，每准备好一个页面按回车截图"
echo ""

screenshots=(
  "01-首页:首页画面，确保最近项目有数据"
  "02-AI对话生成:ChatPage，有对话记录"
  "03-思考链过程:AI正在thinking的状态"
  "04-代码预览-代码Tab:CodePreview代码模式"
  "05-代码预览-预览Tab:CodePreview预览模式"
  "06-模板市场:TemplateMarket，下滑展示模板"
  "07-项目管理:ProjectManage，有多个项目"
  "08-项目详情:ProjectDetail，展开版本历史"
  "09-卡片编辑器:CardEditor，画布上有组件"
  "10-万能卡片桌面:桌面上的2x2卡片"
  "11-个人中心:Profile页"
  "12-设置页:SettingPage"
)

for item in "${screenshots[@]}"; do
  name="${item%%:*}"
  desc="${item#*:}"
  echo "[$name] $desc"
  read -p "准备好后按回车截图..."
  
  hdc shell snapshot_display -f "/data/screen_${name}.png"
  hdc file recv "/data/screen_${name}.png" "$SAVE_DIR/"
  echo "✓ 已保存: $SAVE_DIR/${name}.png"
  echo ""
done

echo "=== 全部截图完成 ==="
ls -la "$SAVE_DIR/"
```

**使用方法**：
```bash
cd ~/DevEcoStudioProjects/HarmonyCraft/参赛资料
chmod +x screenshot.sh
./screenshot.sh
```

## 截图后期处理（可选）

如果截图边缘有多余的系统UI，可用以下命令批量裁剪：

```bash
# 安装ImageMagick（如未安装）
brew install imagemagick

# 批量裁剪状态栏和导航栏（根据实际像素调整）
cd ~/Desktop/HarmonyCraft截图
for f in *.png; do
  # 裁剪掉顶部状态栏(约80px)和底部导航栏(约120px)，保留中间内容
  convert "$f" -crop 1080x2240+0+80 "cropped_$f"
done
```

> 注意：裁剪参数需要根据实际截图分辨率调整。模拟器默认分辨率可能不同，建议先查看一张截图的实际尺寸。

提交前确认以下事项：

- [ ] 12张截图全部拍摄完成
- [ ] 图片格式为PNG，分辨率一致（建议1080×2400）
- [ ] 图片命名规范：`01-首页.png`、`02-AI对话生成.png`...`12-设置页.png`
- [ ] 画面无通知干扰、无加载中状态（03思考链除外）
- [ ] 所有页面有真实数据（非空状态，除非专门展示空状态）
- [ ] 截图保存到 `参赛资料/screenshots/` 目录
