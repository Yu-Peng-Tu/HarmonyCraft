# HarmonyCraft · 鸿蒙AI应用工坊

## 一、项目背景

HarmonyOS生态设备量已突破10亿台，但元服务总量远低于传统应用，开发门槛将大量潜在供给方挡在门外。一位健身教练想创建一个在线预约元服务，在传统路径下需学习ArkTS（2-3周）→掌握ArkUI布局（1-2周）→理解Ability生命周期（1周）→编写万能卡片配置（3-5天）→调试多设备适配（1周），总计6-8周的学习和开发周期。

**HarmonyCraft定位为鸿蒙生态首个对话式元服务生成工具**，Slogan「Speak it, Build it」，让不具备编程能力的用户在5分钟内通过自然语言对话创建一个可运行的鸿蒙元服务。

## 二、核心功能

| 功能模块 | 说明 |
|---------|------|
| **AI对话生成** | 自然语言描述需求 → AI 4步思考链（理解→分析→生成→优化）→ 输出完整ArkTS代码，支持10种场景（计数器/登录/待办/新闻/设置/数据列表/音乐播放器/聊天界面/天气卡片/个人中心） |
| **实时预览引擎** | Tab切换代码/预览双模式，代码区带行号+横向滚动+深色背景，预览区可真实交互（计数、勾选、播放等） |
| **模板市场** | 15个高质量预置模板，7大分类（表单/列表/导航/卡片/弹窗/完整页面），支持一键使用模板进入对话修改 |
| **项目管理** | 项目CRUD、搜索、版本历史查看、重命名、删除，数据通过AppStorage持久化 |
| **万能卡片** | 支持1×2/2×2/2×4三种尺寸添加到桌面，可视化卡片编辑器支持文本/按钮/图片组件叠加编辑 |
| **语音输入** | 支持语音描述需求，模拟ASR语音转文字体验 |

## 三、技术亮点与创新性

### 3.1 形态创新：对话式App生成在鸿蒙生态属首创

DevEco Studio内置的CodeGenie虽有页面生成能力，但产品形态为IDE命令式工具，面向专业开发者。HarmonyCraft首次将Lovable的"对话式全栈生成"产品形态移植到鸿蒙生态，以独立App形态面向非技术用户。

### 3.2 链路创新：端到端AI生成管线

构建"意图理解 → 模板匹配 → ArkTS代码生成 → 实时预览渲染"的全自动化链路：
- **AI思考链4步化**：理解需求→分析布局→生成组件→优化样式，每步带骨架屏动画和时间戳
- **暖色调品牌体系**：琥珀陶土色系（#D4763E/#B85C3D/#FAF6F1）贯穿全应用，区别于市面上千篇一律的蓝紫科技风
- **纯矢量图标体系**：24个统一风格图标，纯Shape+Path代码绘制，1.75px描边圆角端点

### 3.3 集成创新：万能卡片一键生成

从对话生成到万能卡片上架的完整闭环。用户可在对话中指定卡片规格，系统自动完成卡片布局、数据绑定、刷新策略配置，实现传统开发中需手动编写formBindingData和form_config.json的全部工作。可视化卡片编辑器支持多组件叠加和属性实时编辑。

### 3.4 鸿蒙特性深度使用

| Kit/能力 | 应用场景 |
|---------|---------|
| ArkUI声明式UI | 全应用9页+7组件，@State/@Builder驱动响应式界面 |
| FormExtensionAbility | 万能卡片生命周期管理，支持桌面动态刷新 |
| AppStorage | 项目数据、版本历史、用户偏好（深色模式）持久化 |
| Pasteboard Kit | 代码一键复制到剪贴板 |
| Router模块 | 9页间页面跳转与参数传递 |
| 意图框架（模拟） | 模板市场→ChatPage快速启动对话生成 |
