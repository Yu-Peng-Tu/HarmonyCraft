# HarmonyCraft 演示视频脚本（详细版）

> **总时长**：2分20秒
> **分辨率**：1920×1080（横屏，后期将手机录屏嵌入）
> **格式**：MP4，H.264编码
> **选手出镜**：开场10秒 + 结尾10秒，其余画外音

---

## 前期准备

### 数据准备（视频录制前必须完成）

1. **创建3-4个测试项目**：
   - 打开应用 → 进入ChatPage → 依次生成：待办清单、天气卡片、音乐播放器、登录页
   - 每个都点击"导出"保存到项目管理

2. **准备一段完整的对话记录**：
   - 输入："帮我做一个天气卡片，显示城市、温度和天气状况，要好看一点的"
   - 让AI完整走完4步思考链，生成代码
   - 点击"查看预览"进入CodePreview

3. **模板市场收藏1-2个模板**（可选）

4. **清理通知栏**：
   - 模拟器/真机下拉通知栏 → 清除所有通知
   - 确保状态栏干净无干扰图标

## 常见问题

**Q：思考链生成时间太长，怎么压缩？**
> 提前录制好完整的思考链流程，后期剪辑时把4步之间的等待时间从2秒压缩到0.3秒，视觉上仍然流畅。

**Q：没有真机，模拟器能演示万能卡片吗？**
> DevEco Studio 4.0+ 的模拟器支持Form卡片预览。在模拟器上长按桌面空白处即可呼出服务卡片菜单。如不支持，此段可改为纯应用内CardEditor演示。

**Q：台词记不住怎么办？**
> 可以分段录制，每段10-15秒，用提词器App辅助。画外音部分可以后期配音，不要求同期声。

**Q：视频超过3分钟怎么办？**
> 优先压缩镜号03（AI对话生成），思考链加速+去掉部分中间步骤展示。其次是镜号05（模板市场）和镜号07（万能卡片）可适当精简。
