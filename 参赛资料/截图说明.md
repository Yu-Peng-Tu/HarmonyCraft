## 截图拍摄技巧

1. **模拟器截图**：DevEco Studio → Tools → Device Manager → 启动模拟器 → 使用系统截图功能
2. **真机截图**：HarmonyOS三指下滑截屏
3. **画面整洁**：确保状态栏干净（关闭通知图标），画面无遮挡
4. **数据准备**：提前准备一些项目和对话记录，避免空状态
5. **命名规范**：使用`01-`、`02-`前缀确保排序正确
