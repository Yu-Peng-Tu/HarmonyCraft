# HarmonyCraft 演示视频脚本（不出镜版）

> **总时长**：2分10秒
> **形式**：模拟器录屏 + 画外音 + 字幕
> **分辨率**：1080P，MP4，横屏16:9
> **工具**：macOS 自带录屏 (Cmd+Shift+5) + 剪映/手机录音

---

## 快速制作流程（1小时搞定）

1. **录音**（10分钟）：用手机录好画外音台词（见下文）
2. **录屏**（20分钟）：按操作步骤在模拟器上演示一遍
3. **合成**（20分钟）：剪映里拼接画面+添加字幕+导入画外音
4. **导出**（10分钟）：导出MP4

## 剪映操作步骤

### 1. 新建项目
- 打开剪映 → 开始创作 → 选择比例 **16:9**

### 2. 导入素材
- 点击「导入」→ 选择模拟器录屏视频 + 画外音音频文件

### 3. 拼接时间线

按下面的顺序拖入轨道：

| 轨道1（视频） | 轨道2（音频/画外音） | 轨道3（字幕） |
|-------------|-------------------|-------------|
| 开场标题图（10秒） | 开场画外音 | HarmonyCraft · 鸿蒙AI应用工坊 |
| 首页录屏（8秒） | 中段画外音-1 | 10个高频场景快速入口 |
| AI对话录屏（37秒） | 中段画外音-2 | 输入自然语言需求 / AI 4步思考链 / 生成完整ArkTS代码 |
| 代码预览录屏（20秒） | 中段画外音-3 | 代码/预览双模式 |
| 模板市场录屏（17秒） | 中段画外音-4 | 15个预置模板 |
| 项目管理录屏（13秒） | 中段画外音-5 | 版本历史回溯 |
| 卡片编辑器录屏（10秒） | 中段画外音-6 | 万能卡片编辑器 |
| 结尾标题图（15秒） | 结尾画外音 | Speak it, Build it |

### 4. 添加字幕
- 点击「文本」→「识别字幕」（如果画外音清晰，可以自动识别）
- 或者手动在关键时间点添加字幕
- **必须有的字幕**（评审可能静音观看）：
  - "AI 4步思考链"
  - "生成完整ArkTS代码"
  - "15个预置模板"
  - "万能卡片编辑器"

### 5. 添加BGM（可选）
- 点击「音频」→「音乐」→ 选择轻量科技感背景音乐
- 音量调到 **20%-30%**，不要盖过人声

### 6. 导出
- 点击右上角「导出」
- 分辨率：**1080P**
- 帧率：**30fps**
- 格式：**MP4**
- 时长检查：确保 **≤2分30秒**

## 常见问题

**Q：画外音怎么录？**
> 手机打开飞行模式，用语音备忘录App录音。在安静环境，一口气读完一段台词。读错了就重录，后期剪辑拼接。

**Q：模拟器录屏有边框怎么办？**
> Cmd+Shift+5 选「录制所选部分」，只框选手机屏幕区域（不包括模拟器窗口边框和工具栏）。

**Q：视频里要不要加转场效果？**
> 简单淡入淡出即可，不要花哨转场。评审看的是功能，不是特效。

**Q：时长超了怎么办？**
> 压缩AI对话段落（核心但最长），把思考链等待时间剪掉或加速到2倍。其次是模板市场段落。

**Q：不会用剪映怎么办？**
> 下载剪映App，有新手引导。核心操作就3个：拖视频到时间线、加字幕、导出。10分钟就能学会。
