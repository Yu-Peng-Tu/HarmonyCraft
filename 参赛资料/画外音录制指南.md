# 画外音录制指南（最快5分钟搞定）

## 方案A：手机录制（推荐，最简单）

### 步骤

1. **找安静环境**
   - 关窗、关门、关掉空调/风扇
   - 手机开**飞行模式**（防止录音中途来电打断）

2. **打开录音App**
   - iPhone：自带「语音备忘录」
   - 安卓：自带「录音机」或下载「录音大师」

3. **分段录制**
   不要一口气录2分钟，分3段录，出错只重录那一段：

   | 段落 | 时长 | 内容 |
   |------|------|------|
   | 第1段 | 10秒 | 开场 |
   | 第2段 | 1分45秒 | 中段（如果中间读错，从最近一个断句处重新开始） |
   | 第3段 | 15秒 | 结尾 |

4. **朗读技巧**
   - 手机距离嘴巴 **15-20厘米**（不要贴嘴，会喷麦）
   - 语速比平时稍慢，吐字清晰
   - 每段之间留1秒空白（方便后期剪辑）
   - 读错了就暂停，重录这一段

5. **传到电脑**
   - iPhone：语音备忘录 → 点击录音 → 分享 → AirDrop到Mac
   - 安卓：文件管理 → 找到录音文件 → 微信/QQ/数据线传到电脑

---

## 方案B：电脑录制（Mac）

### 步骤

1. **打开「语音备忘录」App**
   - Mac 自带， Launchpad 里找

2. **检查麦克风**
   - 系统设置 → 声音 → 输入 → 选「MacBook麦克风」

3. **开始录制**
   - 点击红色录制按钮
   - 同样分3段录制

4. **导出音频**
   - 右键录音文件 → 「共享」→「存储到文件」
   - 保存为 `.m4a` 格式

## 方案C：剪映直接录音（懒人方案）

如果你用剪映做视频，可以直接在剪映里录音：

1. 剪映导入视频素材后
2. 时间线定位到开头
3. 点击「音频」→「录音」
4. 戴上耳机，点击录制按钮
5. 对着麦克风读台词
6. 读完后点击停止
7. 如果读错了，删掉重来

**缺点**：如果视频很长，要一直盯着画面读，容易分心。
**建议**：先录好音频，再导入剪映对齐画面。

---

## 录音质量检查

录完后听一遍，检查：
- [ ] 没有杂音（空调声、键盘声、窗外车声）
- [ ] 没有喷麦（声音清晰，没有爆破音）
- [ ] 语速适中（不要太快，评审可能听不清）
- [ ] 音量适中（不爆音，也不过小）

**如果音质不好**：找个衣柜/被子围成的小空间录，天然消音室。

---

## 导入剪映

音频文件准备好后（`.m4a` `.mp3` `.wav` 都可以）：

1. 打开剪映
2. 点击「导入」→ 选择音频文件
3. 拖到时间线的音频轨道上
4. 拖动音频，和视频画面对齐

对齐技巧：
- 开场音频对应开场画面
- "点击对话按钮"这句对应AI对话画面开始处
- 如果音频和画面不同步，剪开音频，拖动对齐
